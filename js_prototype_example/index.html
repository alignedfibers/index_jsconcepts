Object Prototype in JS Description
All objects and functions in javascript have a prototype chain, this prototype chain allows objects to adopt or inherit properties and functions from other objects. If a prototype is added to an object then objects of that type will essentially enherit that function or property. The example I have provided here demonstrates adopting prototypes that allow events to be managed from an element object and exposing them within my custom object.
